# Instructions to Run the Pipeline

### 1. Install Dependencies


```bash
uv init
uv add jax dm-haiku numpy

```

### 2. Run the Pipeline

```bash
uv run run_pipeline.py \
  --artifacts_dir ./artifacts/oss-phoenix-artifacts \
  --sequence_file ./artifacts/oss-phoenix-artifacts/example_sequence.json \
  --corpus_file ./artifacts/oss-phoenix-artifacts/sports_corpus.npz \
  --top_k_retrieval 200 \
  --top_k_display 30

```

### 3. View Results
The results will be saved in the `artifacts/oss-phoenix-artifacts` directory. You can view the retrieved documents and their relevance scores in the output files generated by the pipeline.
